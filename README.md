# Smart-Plant-Telemetry
An IoT project consisting of  a esp32 sensor interfaced platform collecting moisture, temperature and humidity values and a web dashboard endpoint that retrieves data from THINGSSPEAK API  to display on the screen . A python script that takes the same data and processes it to give basic statistical valuation on it like mean std dev and so on
